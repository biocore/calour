calour.filtering.filter_by_data¶

calour.filtering.filter_by_data(exp, predicate, axis=0, negate=False, inplace=False, **kwargs)[source]¶

Filter samples or features by data.

Parameters:
  • predicate (str or callable) – The callable accepts a list of numeric and return a bool. Alternatively it also accepts the following strings: ‘sum_abundance’: calls _sum_abundance, ‘freq_ratio’: calls _freq_ratio, ‘unique_cut’: calls _unique_cut, ‘mean_abundance’: calls _mean_abundance, ‘prevalence’: calls _prevalence
  • axis (0 or 1) – Apply predicate on each row (samples) (0) or each column (features) (1)
  • negate (bool) – negate the predicate for selection
  • kwargs (dict) – keyword argument passing to predicate function
Returns:

the filtered object
Return type:

Experiment
